Draws text by calling hw_text_create_surface.

Parameters
surfaceWhere to draw the text. The surface must be locked by hw_surface_lock.
whereCoordinates, in the surface, where to anchor the top-left corner of the rendered text.
textThe string of the text. Can't be NULL.
fontThe font used to render the text. If NULL, the ei_default_font is used.
colorThe text color. The alpha parameter is not used.
clipperIf not NULL, the drawing is restricted within this rectangle.
